# Generic Documentation for eBL

This repository collects technical and editorial documentation for the
[Electronic Babylonian Library (eBL)](https://www.ebl.lmu.de/) platform.
It is the entry point for backend and frontend contributors and editors.

## Primary Documentation Sets

- [Backend (ebl-api)](backend-docs/README.md) — architecture, data model,
  API contract, lifecycle, operations, and contribution rules.
- [Frontend (ebl-frontend)](frontend-docs/README.md) — architecture, data
  and API integration, routes and features, build and operations.
