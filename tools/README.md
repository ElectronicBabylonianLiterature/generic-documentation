# Tools

Created: 2026-05-12

Standalone utilities that support the eBL platform. Each subdirectory keeps
its own README with usage details.

| Directory | Purpose |
| --- | --- |
| [atf-importer-runner/](atf-importer-runner/README.md) | Interactive runner around the ATF importer used to migrate transliterations into the eBL fragmentarium. |
| [cuneiform-tablet-stitch-tools/](cuneiform-tablet-stitch-tools/) | Photoshop actions and JSX scripts used to stitch and scale tablet photographs. See [tutorial/](cuneiform-tablet-stitch-tools/tutorial/) for a worked example. |
